About This Project

This project has been implemented for personal scope. It has nothing to do with any other sources or organizations.

This project has some functionalities which will help the user to track the impact of covid-19 in our world and for each country. Information source is the open NovelCOVID API.
Github: github.com/NovelCOVID/API
